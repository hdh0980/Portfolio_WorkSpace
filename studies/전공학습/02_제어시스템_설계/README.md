# 제어시스템 설계

참고 홈페이지의 `제어 이론` 글 52개를 목차 수준에서 검토한 뒤, 내용을
그대로 옮기지 않고 다음 네 축으로 재구성했습니다.

1. 모델과 피드백 구조를 이해한다.
2. PI·IP·PID 제어기를 설계하고 포화·노이즈 문제를 다룬다.
3. 연속시간 설계를 디지털 제어기로 구현한다.
4. 상태공간·관측기·고급제어로 확장하고 시험으로 검증한다.
