# 전공학습

전력전자·제어·전동화 시스템을 이해하기 위한 학습 자료를 개인적으로 정리하는
공간입니다. 참고 블로그의 카테고리를 그대로 복제하지 않고, 개념의 선수 관계와
나중에 다시 찾아보기 쉬운 개인 학습 순서로 재구성했습니다.

## 학습 체계

```text
전기전자 핵심기초
  → 제어시스템 설계
  → 모델링·시뮬레이션
  → 전력변환 회로
  → 모터구동·센서
  → 전동화 시스템
```

## 작성 원칙

1. 외부 자료는 출처만 참고하고 설명·수식·도식은 직접 재구성합니다.
2. 각 주제는 `개념 → 적용 조건 → 계산·검증 → 학습 기록 → 다음 탐구` 순서로 정리합니다.
3. 실제 회사의 비공개 수치·회로·소스 코드는 공개하지 않습니다.
4. 시뮬레이션과 코드는 실행 조건, 파라미터, 판정 기준을 함께 남깁니다.

작성 현황 표는 실제 파일과 실행 가능한 증거를 기준으로 갱신합니다.
