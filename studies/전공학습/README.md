# 전공학습

전력전자·제어·전동화 시스템 엔지니어의 관점에서 학습 자료를 정리하는
작업 공간입니다. 참고 블로그의 카테고리를 그대로 복제하지 않고, 실제 설계
업무에서 지식이 연결되는 순서로 다시 구성했습니다.

## 학습 체계

```text
전기전자 핵심기초
  → 제어시스템 설계
  → 모델링·시뮬레이션
  → 전력변환 회로
  → 모터구동·센서
  → 전동화 시스템
  → 임베디드 구현과 검증
```

## 카테고리

| 순서 | 영역 | 다룰 내용 |
|---:|---|---|
| 01 | 전기전자 핵심기초 | 회로, 소자, 신호, 측정의 기초 |
| 02 | 제어시스템 설계 | 전달함수, PID, 주파수응답, 상태공간, 검증 |
| 03 | 모델링·시뮬레이션 | MATLAB/Simulink 모델과 해석 절차 |
| 04 | 전력변환 회로 | DC/DC 컨버터와 DC/AC 인버터 |
| 05 | 모터구동·제어 | BLDC/PMSM 구동과 제어 구조 |
| 06 | 센서·신호처리 | 센싱 회로, 필터, ADC, 진단 |
| 07 | 전동화 시스템 | 전기차 기술과 차량 전자제어 |
| 08 | 지능형 모빌리티 | 자율주행과 차량 지능화 |
| 09 | 임베디드 프로토타이핑 | MCU, Arduino, 실시간 구현 |
| 10 | 공학기초·수학 | 선형대수와 기계공학 기초 |
| 11 | 기술동향 리서치 | 기술 뉴스와 공학 상식 |
| 12 | 산업사례 리뷰 | 기업 기술과 제품 사례 |
| 13 | 공학도서 노트 | 서적 기반 학습 기록 |
| 14 | 기술콘텐츠 시리즈 | 전기차와 자동차 기술 연재 |

## 작성 원칙

1. 외부 자료는 출처만 참고하고 설명·수식·도식은 직접 재구성합니다.
2. 각 주제는 `문제 → 원리 → 설계 → 검증 → 결과` 순서로 정리합니다.
3. 실제 회사의 비공개 수치·회로·소스 코드는 공개하지 않습니다.
4. 시뮬레이션과 코드는 실행 조건, 파라미터, 판정 기준을 함께 남깁니다.

현재 작성이 시작된 영역은 [02_제어시스템_설계](./02_제어시스템_설계/)입니다.
