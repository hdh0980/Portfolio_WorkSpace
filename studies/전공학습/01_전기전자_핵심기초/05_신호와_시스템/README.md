# 신호와 시스템

연속·이산 신호와 시스템을 수학적으로 표현하고, 시간영역과 변환영역에서
시스템의 응답과 주파수 특성을 해석하는 기초 과목입니다. 제어 루프 설계,
ADC 샘플링 전략, 디지털 필터 구현의 이론적 토대를 제공합니다.

## 권장 학습 흐름

```text
신호의 표현과 분류 → 시스템의 성질 → LTI시스템과 컨볼루션
  └─→ 푸리에급수 → 푸리에변환 → 라플라스변환과 연속시스템 → 주파수응답과 필터 기초
                              → 샘플링과 복원 → 이산신호와 Z변환 ┘
```

## 제어·파워일렉트로닉스와의 연결

| 신호와 시스템 단원 | 연관 학습 주제 |
|---|---|
| 라플라스변환과 연속시스템 (06) | 전류·전압 루프 전달함수 G(s)·C(s) 설계 |
| 샘플링과 복원 (07) | EV OBC ADC 샘플링 주파수 선정, AAF 필터 τ |
| 이산신호와 Z변환 (08) | 디지털 PI/PID 제어기 구현, 이산화 방법(tustin) |
| 주파수응답과 필터 기초 (09) | 전류·전압 센서 필터 설계, 보드선도 해석 |
| LTI시스템과 컨볼루션 (03) | 고조파 분석 (FFT 기반 THD 측정) |

## 폴더 사용 원칙

1. 연속 신호는 $x(t)$, 이산 신호는 $x[n]$으로 표기합니다.
2. 변환쌍은 $\xleftrightarrow{\mathcal F}$ 또는 $\xleftrightarrow{\mathcal Z}$ 화살표로 나타냅니다.
3. 안정도 판별은 ROC와 극점 위치를 함께 명시합니다.
4. 각 단원의 `학습·검증 기록`에는 다시 확인할 계산 근거와 다음 탐구 항목을 남깁니다.
