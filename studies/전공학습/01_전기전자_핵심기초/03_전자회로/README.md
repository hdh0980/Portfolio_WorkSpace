# 전자회로

반도체 소자의 동작을 이해하고, 바이어스·소신호 모델을 이용해 증폭기와
아날로그 기능 회로를 해석하는 기초 과목입니다. 파워일렉트로닉스에서
MOSFET 스위칭, 게이트드라이버, 전류감지, 보호회로 설계의 기반을 형성합니다.

각 단원은 계산 예제와 표·유도식에서 확인한 근거를 `학습·검증 기록`에 남기고,
나중에 다시 볼 때 이어서 확인할 데이터시트 또는 SPICE 비이상 항목을 함께 적습니다.

## 세부 단원

| 순서 | 단원 | 핵심 학습 내용 |
|---:|---|---|
| 01 | [반도체와 PN접합](./01_반도체와_PN접합/) | 캐리어 농도, 공핍층, 다이오드 I-V 특성 |
| 02 | [다이오드 회로](./02_다이오드_회로/) | 정류, PIV, 클리퍼·클램퍼, 제너, SiC/GaN |
| 03 | [BJT 동작원리](./03_BJT_동작원리/) | 능동·포화·차단, 전류이득 β, 직류 동작점 |
| 04 | [MOSFET 동작원리](./04_MOSFET_동작원리/) | Vth, 선형·포화 영역, 기생 커패시턴스, Rds(on) |
| 05 | [바이어스와 소신호모델](./05_바이어스와_소신호모델/) | gm·ro, 소신호 등가회로, 동작점 안정성 |

| 07 | [연산증폭기](./07_연산증폭기/) | 이상 op-amp, 반전·비반전·차동, 슬루율 |
| 08 | [주파수응답과 피드백](./08_주파수응답과_피드백/) | 밀러 효과, 4종 피드백, 위상·이득 여유 |
| 09 | [발진기](./09_발진기/) | Barkhausen 조건, RC·LC 발진기, PLL 기초 |
| 10 | [출력단과 전원회로](./10_출력단과_전원회로/) | A/B/AB/D급, LDO, 밴드갭, 보호회로 |

## 권장 학습 흐름

```text
반도체와 PN접합
  └─→ 다이오드 회로
        └─→ BJT 동작원리 ─→ 바이어스와 소신호모델 ─→ 증폭기 기본구조 ─→ 주파수응답과 피드백
        └─→ MOSFET 동작원리 ┘                                              └─→ 발진기
                                                                           └─→ 출력단과 전원회로
연산증폭기 (소신호모델 이후 병행 가능)
```

## 파워일렉트로닉스와의 연결

| 전자회로 단원 | 실무 적용 |
|---|---|
| MOSFET 동작원리 (04) | EV OBC 스위칭 소자 선정 · Rds(on) · 게이트 충전 |
| 다이오드 회로 (02) | SiC PFC 다이오드, 역병렬 다이오드, 스너버 |
| 연산증폭기 (07) | 전류감지 증폭기(INA), 오차증폭기 |
| 주파수응답과 피드백 (08) | 전류·전압 루프 보상기(type-II/III) |
| 출력단과 전원회로 (10) | 게이트드라이버 바이어스, LDO, 과전류 보호 |

## 폴더 사용 원칙

1. 단원 폴더의 `README.md`에서 학습 목표와 핵심 수식을 확인합니다.
2. 회로도는 교재 캡처 이미지로 제공하며 출처를 명시합니다.
3. 소신호 해석과 대신호 동작을 구분해 설명합니다.
