# 논리회로

이진 정보를 논리식으로 표현하고, 조합·순차논리와 상태기계를 이용해
디지털 시스템의 기본 구조를 설계하는 과목입니다. MCU 주변장치(GPIO·타이머·SPI·CAN)
이해와 FPGA 기반 제어기 구현의 토대를 제공합니다.

## 권장 학습 흐름

```text
수체계와 디지털코드
  └─→ 부울대수와 논리게이트 → 논리식 간소화 → 조합논리회로
                                                    └─→ 플립플롭 → 레지스터와 카운터
                                                                      └─→ 순차논리와 상태기계
메모리와 프로그램가능논리 (FPGA: 조합·순차 이후)
타이밍과 HDL 기초 (병행 가능)
```

## 임베디드·MCU와의 연결

| 논리회로 단원 | 연관 학습 주제 |
|---|---|
| 수체계와 디지털코드 (01) | MCU 레지스터 비트 조작, CAN 프레임 파싱 |
| 플립플롭 (05) | 클록 도메인 크로싱, 동기화 체인 |
| 순차논리와 상태기계 (07) | AUTOSAR RunEntity 상태 설계, 충전 시퀀스 FSM |
| 메모리와 프로그램가능논리 (08) | AURIX PFLASH 구조, FPGA 프로토타입 |
| 타이밍과 HDL 기초 (09) | IP 코어 통합, FPGA 타이밍 클로저 |

## 폴더 사용 원칙

1. 진리표·상태표는 마크다운 표로 작성합니다.
2. 논리식은 부울 표기($A \cdot B$, $\overline{A}$)를 사용합니다.
3. HDL 코드 예제는 합성 가능(synthesizable) 스타일로 제한합니다.
4. 각 단원은 계산·설계 근거와 다음 탐구를 `학습·검증 기록`에 남깁니다.
