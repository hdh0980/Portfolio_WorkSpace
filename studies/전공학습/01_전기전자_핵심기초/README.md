# 전기전자 핵심기초

전기전자공학 전공과정의 출발점이 되는 다섯 과목을 학습 단위별로 정리하는
공간입니다. 수학·물리 선수과목은 포함하지 않으며, 제어·전력변환·모터·센서
등의 심화 영역은 `전공학습` 아래의 별도 폴더에서 다룹니다.

개인 재학습 기록으로서 각 단원은 재열람을 위해 핵심 정리·확인 근거·다음 탐구를 남깁니다.

## 학습 영역

| 순서 | 과목 | 핵심 학습 범위 |
|---:|---|---|
| 01 | [회로이론](./01_회로이론/) | 회로 법칙, 회로망 해석, 과도응답, 교류와 주파수응답 |
| 02 | [전자기학](./02_전자기학/) | 전계·자계, 물질 경계, 맥스웰 방정식, 전자파와 전송선 |
| 03 | [전자회로](./03_전자회로/) | 반도체 소자, 증폭기, 연산증폭기, 피드백과 발진 |
| 04 | [논리회로](./04_논리회로/) | 부울대수, 조합·순차논리, 상태기계, 메모리와 HDL 기초 |
| 05 | [신호와 시스템](./05_신호와_시스템/) | LTI 시스템, 컨볼루션, 변환 해석, 샘플링과 필터 기초 |

## 권장 학습 흐름

```text
회로이론
  ├─→ 전자기학
  ├─→ 전자회로
  ├─→ 논리회로
  └─→ 신호와 시스템
```

회로이론을 먼저 학습한 뒤 나머지 과목을 병행할 수 있습니다. 전자회로는
회로 해석을, 신호와 시스템은 함수·복소수·미적분의 기본 활용을 전제로 합니다.

## 폴더 사용 원칙

1. 과목 폴더의 `README.md`에서 학습 목표와 세부 단원을 확인합니다.
2. 학습 자료는 해당 세부 단원 폴더에 추가합니다.
3. 설명은 `개념 → 수식 → 적용 조건 → 계산·검증 → 학습 기록` 순서로 작성합니다.
4. 외부 자료를 참고할 때는 출처를 명시하고 본문과 도식을 직접 재구성합니다.
5. 제어·전력전자 등 심화 내용은 이 폴더에 중복 작성하지 않고 관련 영역으로 연결합니다.
