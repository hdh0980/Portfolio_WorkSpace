# 전자기학

전하와 전류가 만드는 전계·자계를 공간적으로 해석하고, 시변 전자계와
전자파로 확장하는 기초 과목입니다. 파워일렉트로닉스에서 자기회로·인덕터·변압기
설계의 핵심 이론적 토대를 제공합니다.

## 권장 학습 흐름

```text
벡터해석과 좌표계
  ├─→ 정전계 → 전위와 정전용량 → 도체·유전체와 경계조건
  ├─→ 정상전류와 전도
  ├─→ 정자계 → 자성체와 인덕턴스
  └─→ 시변전자계와 맥스웰 방정식 → 전자파와 전송선
```

벡터해석을 먼저 익힌 뒤 정전계·정자계를 병행하고, 맥스웰 방정식으로
통합한 다음 전자파로 확장하는 순서를 권장합니다.
